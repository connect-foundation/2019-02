## 🔨 Usage
### 프로젝트 구성도
```bash
📦 .github/workflows             # CI/CD
📦 web                           # 프론트 서버
📦 server-api                    # API 서버
📦 server-converter              # 이미지 컨버터 서버
```

### 프로젝트 시작하기
> 각 폴더 별로 실행 시켜야 합니다.
#### 1. web
```bash
cd web
yarn install                     # 모듈 설치
yarn dev                         # 개발모드 시작
```

#### 2. server-api
```bash
cd server-api
yarn install                     # 모듈 설치
yarn dev                         # 개발모드 시작
```

#### 3. server-converter
```bash
brew install graphicsmagick gs   # OS 패키지 설치

cd server-converter
yarn install                     # 모듈 설치
yarn dev                         # 개발모드 시작
```
